# spring-board-crud

### 프로젝트 개요

Java를 공부하며 간단한 프로젝트를 만들어보기 위해 게시판 서비스를 만들었습니다.

#### 구현사항 

- 게시글 CRUD
- 페이지네이션
- 게시글 필터

#### 스택
  Java8, Spring Boot2.x, MySQL, Gradle
